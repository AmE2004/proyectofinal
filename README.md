# proyectofinal
## Instalación y configuración
Para utilizar este repositorio se requiere:
- Contar con una cuenta en GitHub
- Tener Git instalado en el equipo
- Un entorno de desarrollo compatible con Java (por ejemplo, IntelliJ IDEA)

## Uso del programa
Este repositorio se utilizará para el desarrollo de un proyecto en Java.
Los cambios en el código se registrarán mediante el uso de ramas en Git,
siguiendo una estructura de trabajo con las ramas master y develop.
